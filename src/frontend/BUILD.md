# Building Nothing Player for Android (Offline APK)

This guide explains how to build an offline Android APK that bundles the entire web application for use without an internet connection.

## Prerequisites

### Web Build Requirements
- Node.js 18+ and pnpm
- DFX (Internet Computer SDK) for backend deployment

### Android Build Requirements
- Android Studio (latest stable version)
- Android SDK (API level 24 or higher)
- Gradle 8.0+
- JDK 17+
